# AP Chemistry — Explained &amp; Interactive

A single-page AP Chemistry guide that pairs **plain-language explanations** with
**40+ hands-on widgets** and diagrams. No build step, no dependencies — open
`index.html` in any browser.

**Live site:** https://AbyzAnna.github.io/ap-chem-explained/

## What's inside

Every topic gets the same treatment: plain-words idea → real-life analogy →
deeper "why" → an interactive widget you can drag, type, or click.

### Unit 1 · Measurement
- **Sig-fig challenge** — random numbers, streak counter
- **Scientific-notation converter** — type any number, see it fold
- **Dartboard simulator** — visualize accurate vs precise
- **Drag-and-drop dimensional analysis** — set up conversion factors so units cancel

### Unit 2 · Matter &amp; the Atom
- **Matter sorter** — drag items into element / compound / homogeneous / heterogeneous
- **Physical vs chemical change identifier** — quick-fire tagging game
- **Atomic-model timeline** — animated Dalton → Thomson → Rutherford → Bohr → Schrödinger
- **Isotope builder** — add/remove protons &amp; neutrons, watch element &amp; isotope label update
- **Bohr atom simulator** — excite electrons, watch them drop &amp; emit photons; lines collect in a spectrum bar
- **Electron-configuration builder** — pick any Z (1–36), see the orbital diagram fill following Aufbau + Hund; auto-flags Cr/Cu exceptions
- **Quantum-number validator** — instantly catches illegal n/ℓ/mℓ combos
- **Average atomic mass calculator** — two-isotope weighted average

### Unit 3 · Periodic Table
- **Interactive periodic table** (Z 1–36) — click any element; shade by block, radius, IE, EN, or metal/nonmetal
- **Trend visualizer** — animated arrow shows where each trend grows
- **Photon calculator** — slide λ across the EM spectrum; f, E (J), E (eV), and region all live

### Unit 4 · Bonding
- **Bond-type classifier** — pick any two elements, see ΔEN + verdict
- **Naming game** — match formula ↔ name
- **VSEPR explorer** — rotating 3-D molecules (CH₄, H₂O, NH₃, CO₂, BF₃, SF₆, PCl₅)
- **Polarity visualizer** — bond dipole arrows + net dipole
- **IMF identifier** — pick the strongest force in each molecule

### Unit 5 · Reactions
- **Reaction-type sorter** — synthesis/decomp/SR/DR/combustion
- **Live equation balancer** — type coefficients, atom counter updates, locks green when balanced
- **Oxidation-number solver** — type the value, instant verdict + explanation

### Unit 6 · Stoichiometry
- **Mole-map walkthrough** — pick a reaction; gram-to-gram with every step shown
- **Limiting-reactant calculator** — finds limiter + leftover automatically

### Unit 7 · Acids &amp; Bases
- **Dilution calculator (M₁V₁ = M₂V₂)** — leave any box blank
- **pH playground** — slider + color-changing beaker, [H⁺], [OH⁻], pOH

### Unit 8 · Gases
- **KMT particle simulator** — temperature + piston sliders, particles speed up &amp; bounce
- **Combined gas-law solver** — leave any of P₁/V₁/T₁/P₂/V₂/T₂ blank, tool solves

### All Units · Practice
- **35+ self-grading multiple-choice questions** with per-unit filters and explanations

## Run locally

```bash
open index.html
```

That's it. No npm, no build, no server. Everything is one file.
